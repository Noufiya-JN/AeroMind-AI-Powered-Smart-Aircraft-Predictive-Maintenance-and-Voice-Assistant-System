# AeroMind-AI-Powered-Smart-Aircraft-Predictive-Maintenance-and-Voice-Assistant-System
AeroMind is an AI-powered aircraft predictive maintenance system that uses the NASA CMAPSS dataset and XGBoost to predict Remaining Useful Life (RUL) of aircraft engines. Built with Streamlit, it includes real-time engine health prediction, interactive dashboards, and a voice-enabled AI assistant.
